ルアー図鑑

ハードベイトからソフトベイトまで、主要なルアータイプの特徴・使いどころ・定番アイテムを一覧で解説。フィールドでの選択に迷ったときの指針になるはずだ。

トップ/ルアー図鑑

HARD BAITハードベイト

ウォブリング

クランクベイト

丸みを帯びたボディとリップによる潜行が特徴。巻くだけで一定のレンジをトレースでき、障害物へのコンタクトで生まれるイレギュラーアクションがバイトを誘発する。初心者でも扱いやすく、春のシャローから夏のディープまで季節を問わない万能選手。

状況
カバー周り / リップラップ / 巻き物で広く探りたいとき
定番: O.S.P ブリッツ / メガバス SR-X / ダイワ ピーナッツII
ウォブンロール

ミノー

細身のボディで小魚を忠実に模倣するルアーの王道。ただ巻き、トゥイッチ、ジャークと多彩な操作に対応し、シーバスからトラウトまでターゲットを選ばない。フローティング・サスペンド・シンキングの使い分けがキモ。

状況
ベイトフィッシュが見えるとき / クリアウォーター / 表層付近
定番: メガバス VISION ONETEN / ラッキークラフト ポインター / シマノ サイレントアサシン
タイトバイブレーション

バイブレーション

ヘッド部分でラインアイが上に位置し、小刻みな振動で広範囲にアピールする。飛距離が出やすく、サーチベイトとしてのパイロットルアーに最適。リフト&フォールでリアクションバイトも狙える。冬場のディープ攻略にも欠かせない。

状況
広いフラットエリア / 低水温期 / 風が強い日
定番: ジャッカル TN60 / ダイワ T.D.バイブレーション / エバーグリーン ブザービーター
フラッシング+バイブレーション

スピナーベイト

ワイヤーアームにブレードとスカート付きヘッドを組み合わせた独特の形状。根がかりしにくく、濁りの中でもフラッシングと振動で存在感を示す。スローロールからバジングまで、リトリーブスピードで異なるレンジを攻められる。

状況
マッディウォーター / カバー周辺 / 風が吹いて水面が荒れたとき
定番: O.S.P ハイピッチャー / ノリーズ クリスタルS / エバーグリーン D-ZONE
強バイブレーション

チャターベイト

ブレーデッドジグとも呼ばれ、金属プレートが水を受けて不規則に振動する。スピナーベイトほど根がかりに強くないが、より強い波動とタイトなアクションが持ち味。プリスポーンからアフターの時期に、ウィードエリアのただ巻きで威力を発揮する。

状況
ウィードの上 / プリスポーン期 / スピナーベイトで反応が薄いとき
定番: エバーグリーン ジャックハンマー / O.S.P パワーダンク / ノリーズ フラチャット
サーフェスクラッター

バズベイト

水面をプロペラ状のブレードでかき回しながら引くトップウォータールアー。着水後すぐに巻き始めるのがコツで、バスが水面を割って出る瞬間は釣りの醍醐味そのもの。朝夕のローライト時や、カバーの際をかすめるように引くのが効果的。

状況
朝夕のマズメ / カバー際 / 水温20度以上の活性が高いとき
定番: ゲーリーヤマモト バズベイト / ノリーズ ボルケーノグリッパー / deps バズジェット
ポップ / スプラッシュ

ポッパー

カップ状のマウスが水を噛み、「ポコッ」という捕食音とスプラッシュを発生させる。移動距離を抑えた誘いが可能で、ピンスポットで粘りたい状況に強い。バスだけでなく、チヌやシーバスのトップゲームでも定番のルアータイプ。

状況
朝夕 / ストラクチャー際 / 魚が上を意識しているとき
定番: メガバス POP-X / ダイワ スティーズポッパー / ラッキークラフト サミー
ドッグウォーク

ペンシルベイト

リップを持たない棒状のトップウォータールアー。ロッド操作で左右に首を振る「ドッグウォーク」アクションが基本。水面に逃げ惑う小魚を演出し、チェイスからバイトまでの一部始終が見える興奮がある。シーバスのボイル撃ちにも有効。

状況
ボイル発生時 / ベイトが水面に追われているとき / 凪
定番: ヘドン ザラスプーク / メガバス DOG-X / ジャッカル ライザーベイト
ドッグウォーク / ポッピング

フロッグ

中空ボディにダブルフックを内蔵した、ヘビーカバー攻略の切り札。マットカバーの上をスキッピングさせたり、オープンウォーターでドッグウォークさせたりと使い方は幅広い。フッキングにはPEラインとパワーのあるロッドが不可欠。

状況
ヒシモ・リリーパッド / マットカバー / 夏のシャロー
定番: deps スリザーク / エバーグリーン キッカーフロッグ / ジャッカル カエラ
S字 / ウォブリング

スイムベイト

ジョイントボディやソフトテールで、まるで本物の魚が泳いでいるかのようなリアルな波動を出す大型ルアー。デカバス狙いの切り札として、琵琶湖やリザーバーで多くの実績を持つ。デッドスローからミディアムリトリーブが基本。

状況
ビッグフィッシュ狙い / クリアウォーター / ベイトの群れがいるエリア
定番: deps スライドスイマー / ガンクラフト ジョインテッドクロー / イマカツ レプリケーター
フォール / ジャーク

メタルジグ

金属の塊をベースにした高比重ルアー。圧倒的な飛距離とフォールスピードが武器で、ショアジギングからオフショアジギングまで対応範囲は広い。ワンピッチジャーク、ただ巻き、フォール主体と、ジャンルによってアクションを使い分ける。

状況
回遊魚狙い / ディープ攻略 / 飛距離が必要なサーフ
定番: シマノ コルトスナイパー / ダイワ サムライジグ / メジャークラフト ジグパラ

SOFT BAITソフトベイト / ワーム系

ボトムバンプ / スイミング

ワーム（テキサスリグ）

バレットシンカーとオフセットフックの組み合わせで、カバーへの貫通力とスナッグレス性能を両立させた基本リグ。ホグ系・クロー系ワームとの相性が抜群で、カバー撃ちの定番。シンカーのウエイトでフォールスピードを調整する。

状況
レイダウン / ブッシュ / テトラ周辺のカバー撃ち
定番: O.S.P ドライブクロー / deps ブルフラット / ゲーリー ファットイカ
ヘッドダウン / シェイク

ワーム（ネコリグ）

ストレートワームの頭にネイルシンカーを埋め込み、ワッキー掛けでセットする日本発祥のフィネスリグ。ボトムで頭を下にした姿勢からシェイクすると、まるでエビが跳ねるようなアクションを生み出す。タフコンディションの最終兵器。

状況
タフコンディション / プレッシャーの高いフィールド / 見えバス
定番: ジャッカル フリックシェイク / レイドジャパン ファットウィップ / O.S.P HP3Dワッキー
中層シェイク / ホバスト

ワーム（ダウンショットリグ）

フックの下にリーダーを介してシンカーを配置し、ワームをボトムから一定の高さで漂わせるリグ。ボトムの起伏に合わせてワームが自然に揺れ、スレた魚にも口を使わせる。リーダーの長さで攻めるレンジを細かく調整できるのが強み。

状況
ピンスポット / 縦ストラクチャー / ディープの魚
定番: レインズ スワンプ / ジャッカル クロステールシャッド / ケイテック スイングインパクト
ダート / フォール

エギ

エビを模した日本古来のルアーで、アオリイカをはじめとするイカ類を狙う専用設計。シャクリで跳ね上げ、フォールで抱かせるのが基本操作。号数（サイズ）と沈下スピード、カラーの組み合わせが釣果を大きく左右する。

状況
春の親イカ / 秋の新子 / 藻場・岩礁帯周辺
定番: ヤマシタ エギ王K / ダイワ エメラルダス ピーク / デュエル EZ-Qキャスト
