釣り場ガイド

全国の人気フィールドを地域別に紹介。対象魚やベストシーズン、アクセス情報など釣行計画に必要な情報をまとめています。

霞ヶ浦 茨城県
対象魚ブラックバス / ワカサギ / ナマズ
ベストシーズン4月〜10月
アクセス常磐道 土浦北ICから約15分
駐車場各ポイントに無料Pあり

日本第2位の面積を誇る湖。おかっぱりポイントが豊富でバスフィッシングの聖地として全国から釣り人が集まる。

東京湾奥 千葉/東京
対象魚シーバス / クロダイ / タチウオ
ベストシーズン9月〜12月
アクセス湾岸線各IC / 電車でも可
駐車場周辺コインP

都心から近く仕事帰りのナイトゲームが人気。秋のコノシロパターンではランカーシーバスの実績多数。

相模湾 神奈川県
対象魚キハダマグロ / カツオ / シイラ
ベストシーズン7月〜10月
アクセス小田原厚木道路 平塚ICから約20分
駐車場各港に有(有料あり)

夏場のキハダゲームは関東屈指の人気。相模湾一帯の船宿からオフショアジギング・キャスティングが楽しめる。

琵琶湖 滋賀県
対象魚ブラックバス / ビワマス
ベストシーズン3月〜11月
アクセス名神高速 瀬田西ICから約10分
駐車場各エリアに無料Pあり

日本最大の湖にしてバスフィッシングの最高峰フィールド。ロクマル以上の実績が毎年多数報告されている。

淡路島 兵庫県
対象魚アオリイカ / メバル / 青物
ベストシーズン9月〜12月(エギング) / 通年
アクセス神戸淡路鳴門道 淡路ICから各港
駐車場有(漁港により異なる)

島全域が釣りスポット。秋のアオリイカ、冬のメバリングが特に人気で、関西圏からのアクセスも良好。

紀伊水道 和歌山県
対象魚マダイ / 青物 / アオリイカ
ベストシーズン4月〜6月 / 9月〜11月
アクセス阪和道 海南ICから約30分
駐車場各漁港に無料Pあり

潮通しの良い好漁場。タイラバやジギングのオフショアに加え、地磯からのショアジギングも盛ん。

玄界灘 福岡県
対象魚ヒラマサ / ブリ / キハダマグロ
ベストシーズン5月〜11月
アクセス福岡都市高速 姪浜出口から各港
駐車場各港に有(無料)

ヒラマサジギングの聖地。10kg超えのヒラマサが狙え、秋にはキハダマグロも回遊する日本屈指の好漁場。

錦江湾 鹿児島県
対象魚カンパチ / マダイ / アオリイカ
ベストシーズン6月〜11月
アクセス九州道 鹿児島ICから約20分
駐車場各港に有

桜島を望む内湾ながら水深が深く、カンパチやマダイの大物が狙える。ティップランエギングも人気。

遠賀川 福岡県
対象魚ブラックバス / ナマズ / シーバス
ベストシーズン4月〜10月
アクセス九州道 八幡ICから約15分
駐車場河川敷に無料Pあり

九州を代表するバスリバー。トーナメントも頻繁に開催され、河口域ではシーバスも好実績。

支笏湖 北海道
対象魚ブラウントラウト / ニジマス / ヒメマス
ベストシーズン5月〜10月
アクセス新千歳空港から車で約40分
駐車場有(有料 500円/日)

透明度日本一クラスの湖。大型ブラウントラウトの実績があり、ウェーディングでのルアー釣りが主流。

十勝川 北海道
対象魚アメマス / ニジマス / サケ(規制有)
ベストシーズン6月〜10月
アクセス道東道 音更帯広ICから各ポイント
駐車場河川敷に無料Pあり

北海道を代表する大河川。本流でのニジマスは60cmクラスも珍しくなく、ネイティブトラウトの宝庫。

積丹半島 北海道
対象魚ヒラメ / ソイ / ホッケ / ブリ
ベストシーズン5月〜10月(青物) / 通年
アクセス札樽道 小樽ICから約60分
駐車場各漁港に無料Pあり

積丹ブルーの海が広がる道央屈指の磯釣りエリア。近年はブリのショアジギングが注目を集めている。

浜名湖 静岡県
対象魚クロダイ / シーバス / マゴチ
ベストシーズン4月〜11月
アクセス東名高速 浜松西ICから約15分
駐車場各ポイントに無料Pあり

汽水湖ならではの豊富な魚種。チニングの聖地として知られ、トップウォーターでのクロダイゲームが盛ん。

伊勢湾 三重県
対象魚マダイ / 青物 / タチウオ
ベストシーズン5月〜12月
アクセス伊勢道 伊勢ICから各港へ
駐車場各港に有

タイラバ発祥の地ともいわれるエリア。大型真鯛の実績が高く、秋冬のタチウオジギングも人気。

瀬戸内海 広島県
対象魚マダイ / メバル / アコウ
ベストシーズン通年(春秋がベスト)
アクセス山陽道 広島ICから各港へ
駐車場有(港により異なる)

穏やかな内海に多島美が広がる。潮流の変化が豊富で、タイラバ・メバリング・アコウゲームの好フィールド。

四万十川 高知県
対象魚アユ / アカメ / テナガエビ
ベストシーズン6月〜9月(アユ) / 通年(アカメ)
アクセス高知道 四万十町中央ICから約40分
駐車場各沈下橋周辺に無料Pあり

日本最後の清流と称される名河川。幻の魚アカメが棲息し、夏場は天然アユの友釣りで賑わう。
