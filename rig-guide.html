リグ（仕掛け）図鑑

同じワームでもリグが変われば動きは一変する。テキサスからフリーリグまで、現場で迷わないためのリグ辞典。必要パーツ、セッティング手順、最適なシチュエーションを網羅した。

トップ/リグ（仕掛け）図鑑
01

テキサスリグ

バレットシンカーをラインに通し、オフセットフックにワームをセットするカバー撃ちの王道リグ。シンカーが先にカバーに入り込むことで根がかりを回避しつつ、奥までワームを送り込める。ペグ止めの有無でシンカーの遊動幅を変え、フォールの質を調整する。

必要パーツ

  • バレットシンカー（3.5g - 14g）
  • オフセットフック（#1/0 - #5/0）
  • ワーム（ホグ系・クロー系）
  • ペグ止め（任意）

有効な状況

レイダウン、ブッシュ、アシ際などのカバー撃ち全般。特に濃いカバーの奥へ入れたいときに真価を発揮する。フリッピング・ピッチングとの相性が良い。

セッティング手順

  1. ラインにバレットシンカーを通す（先端が細い方をロッド側に向ける）
  2. オフセットフックを結ぶ（パロマーノットまたはクリンチノット）
  3. 必要に応じてシンカーとフックの間にペグ止めを入れる
  4. フックポイントをワームの頭からまっすぐ刺し、クランク部分で折り返す
  5. ワームのボディにフックポイントを埋め込み、針先が出ないようセットする
02

キャロライナリグ

シンカーとワームの間にリーダーを設けることで、ボトムをシンカーが叩きながらワームがフワフワと漂う独特のアクションを生み出す。広いフラットエリアをスローに探るのに適しており、ズル引きでボトムの変化を感じ取りながらバイトを待つ釣り方が基本。

必要パーツ

  • バレットシンカー（7g - 21g）
  • ビーズ（ガラスまたはプラ）
  • スイベル
  • リーダー（30 - 90cm）
  • オフセットフック
  • ワーム（ストレート・リザード系）

有効な状況

広いフラットエリア、ハンプ周辺、ウィードエッジなど、面で広く探りたい場面。水温が上がりきらない春先や秋のターンオーバー時期にも有効。

セッティング手順

  1. メインラインにバレットシンカーを通す
  2. ビーズを通す（シンカーとの衝突音でアピール）
  3. スイベルを結ぶ
  4. スイベルのもう片方にリーダー（フロロ 30-90cm）を結ぶ
  5. リーダーの先にオフセットフック + ワームをセット
03

ダウンショットリグ

フックの下にリーダーを取り、その先端にシンカーを付けるリグ。ワームがボトムから浮いた状態をキープでき、シェイクで細かくアクションさせると高いフィネス効果を発揮する。リーダーの長さでワームの浮き具合を調整し、魚のレンジに合わせる。

必要パーツ

  • ダウンショットシンカー（1.8g - 5g）
  • マスバリまたは細軸オフセット（#1 - #2）
  • ワーム（ストレート・シャッドテール系）

有効な状況

ピンスポットを丁寧に攻めたいとき。縦ストラクチャー（護岸・杭・立木）沿いのシェイクや、ディープエリアでの定点攻撃に強い。トーナメントでも多用されるフィネスリグの代表格。

セッティング手順

  1. フックを結ぶ際、余り糸を長め（15-40cm）に残す
  2. パロマーノットで結び、余り糸をフックアイのフック先端側に通す
  3. 余り糸の先端にダウンショットシンカーをセット
  4. フックにワームをちょん掛けまたはワッキー掛けする
04

ネコリグ

「根こそぎ釣れるリグ」の略で、日本で開発されたフィネステクニック。ストレートワームの頭にネイルシンカーを挿入し、ボディ中央付近をワッキー掛けにする。ボトムで頭を下にした独特の姿勢からシェイクすると、エビが跳ねるような動きを見せる。

必要パーツ

  • ネイルシンカー（0.45g - 1.8g）
  • マスバリまたはガード付きフック（#4 - #1）
  • ストレートワーム（5 - 6.5インチ）
  • ワームチューブ（任意・ワーム保護用）

有効な状況

霞ヶ浦などハイプレッシャーフィールドの定番。ハードボトム上のシェイク、護岸際の縦の釣り、沈み物周りなど汎用性が極めて高い。タフコンディション時の最終手段。

セッティング手順

  1. ストレートワームの頭（太い方）からネイルシンカーを挿入する
  2. シンカーが飛び出さないよう、奥まで押し込む
  3. ワームのちょうど中央付近にマスバリをチョン掛けする
  4. カバー周りではガード付きフックやワッキーチューブを使用
05

ジグヘッドワッキー

ジグヘッドをワームの中央にワッキー掛けするリグ。ネコリグに似ているが、ジグヘッドの重心がフック位置にあるため水平フォールしやすく、沈下中の自発的なアクション（クネクネ）がバイトを誘う。中層のフォーリングで食わせるイメージ。

必要パーツ

  • ワッキー用ジグヘッド（0.9g - 2.7g）
  • ストレートワーム（4 - 5.5インチ）
  • ワームチューブ（任意）

有効な状況

中層にサスペンドした魚を狙うとき。オープンウォーターのフォーリング、護岸のカーブフォール、シェードの下にワームを送り込む際に有効。

セッティング手順

  1. ワームの中央（ハチマキ部分）にジグヘッドを横刺しにする
  2. フックポイントをワームの反対側から抜く
  3. ワームが水平にバランスするよう位置を微調整する
06

ノーシンカーリグ

シンカーを一切使わず、フックとワームの自重だけでアクションさせるリグ。ゆっくりとしたフォールと自然な動きが最大の武器で、ハイプレッシャー下でもバスの警戒心を解く。ファットなワームほど自重が出るため、飛距離を確保しやすい。

必要パーツ

  • オフセットフック（#1/0 - #4/0）
  • ワーム（ファットイカ・グラブ・スティックベイト等）

有効な状況

クリアウォーター、プレッシャーの高い人気ポイント、サイトフィッシング。カバー際へのフォーリングや、水面直下のスイミングにも使える。

セッティング手順

  1. オフセットフックをラインに結ぶ
  2. ワームの頭からフックを真っ直ぐ刺す
  3. クランク部で折り返し、ボディに針先を埋め込む
  4. ワームが真っ直ぐになるようセットし、回転を防ぐ
07

スプリットショットリグ

フックの30-50cm上にガン玉（スプリットショット）をかませるだけのシンプルなリグ。キャロライナリグの簡易版ともいえるが、より軽量なシンカーでナチュラルに漂わせられるのが利点。スモールマウスバスや低活性時に効果的なスローフィネスの選択肢。

必要パーツ

  • ガン玉（BB - 3B程度）
  • マスバリまたは小型オフセット
  • ワーム（ストレート・小型グラブ）

有効な状況

スモールマウスバスのボトムズル引き、クリアウォーターでのフィネスアプローチ、管理釣り場のトラウトにも応用可。

セッティング手順

  1. フックを結んでワームをセットする
  2. フックから30-50cm上のラインにガン玉をかませる
  3. ペンチでしっかり固定する（締めすぎるとラインが傷むので注意）
08

フリーリグ

韓国発祥で近年日本でも急速に普及したリグ。スティック型のフリーリグシンカーを遊動式で使い、着底後にシンカーだけが先に沈み、ワームがフリーフォールする時間を作り出す。テキサスリグの貫通力とノーシンカーのナチュラルさを兼ね備える。

必要パーツ

  • フリーリグシンカー（3.5g - 10g）
  • オフセットフック（#1/0 - #3/0）
  • ワーム（ホグ系・クロー系・シャッドテール）

有効な状況

ロックフィッシュ（アコウ・カサゴ）のボトム攻略、バスのカバー撃ち、チニングのボトムゲームなど。汎用性の高さが最大の魅力。

セッティング手順

  1. ラインにフリーリグシンカーを通す（ラインアイがある方がロッド側）
  2. オフセットフックを直結する（ペグ止めは基本的に使わない）
  3. ワームをまっすぐセットする
09

ヘビキャロ（ヘビーキャロライナリグ）

キャロライナリグの重量版で、21g以上のシンカーとロングリーダーで広大なエリアをサーチする。琵琶湖のウィードフラットで発達したテクニックで、ロングキャストからのズル引きで沖のブレイクやウィードエッジを効率よく探る。竿抜けポイントを直撃できる。

必要パーツ

  • 中通しシンカー（21g - 42g）
  • ビーズ
  • スイベル
  • リーダー（60 - 150cm / フロロ）
  • オフセットフック
  • ワーム

有効な状況

琵琶湖のウィードフラット、広大なリザーバーのディープフラット。おかっぱりから沖のブレイクラインを攻める際にも重宝する。夏から秋にかけてが最盛期。

セッティング手順

  1. メインライン（PE 1-2号推奨）に中通しシンカーを通す
  2. ビーズを通し、スイベルを結ぶ
  3. スイベルにフロロリーダー（60-150cm）を結ぶ
  4. リーダー先端にオフセットフック+ワームをセット
  5. 7フィート以上のヘビーロッドでフルキャストする
10

ジカリグ（直リグ / パンチショット）

フックのアイにスプリットリングやスイベルを介してシンカーを直接ぶら下げるリグ。テキサスリグより垂直にフォールしやすく、ピンスポットへの精度が高い。ボトムでの姿勢が安定し、シェイクした際のワームのアクションも大きく出る。

必要パーツ

  • リング付きシンカー（3.5g - 10g）
  • スプリットリング
  • オフセットフック
  • ワーム（ホグ系・クロー系）

有効な状況

消波ブロック（テトラ）の穴撃ち、護岸際のリアクション、垂直にフォールさせたい縦の釣り全般。テキサスリグではシンカーが先行しすぎる場面で活躍。

セッティング手順

  1. オフセットフックのアイにスプリットリングを通す
  2. スプリットリングにリング付きシンカーを接続する
  3. フックにワームをセットする
  4. ラインをフックアイに直結する
11

ウェイテッドフック（ウェイテッドオフセット）

フックのシャンク部分にウエイトが一体化した、または後付けしたフック。テキサスリグからシンカーを省いた簡素な構成でありながら、適度なウエイトによるフォールスピードとスナッグレス性能を確保する。スイムベイトやシャッドテールワームとの相性が抜群。

必要パーツ

  • ウェイテッドフック（1/16oz - 3/8oz）
  • ワーム（シャッドテール・スイムベイト系）

有効な状況

シャッドテールワームのスイミング、ウィード上のただ巻き、中層からボトム付近のスローリトリーブ。パーツが少なくトラブルも少ない。

セッティング手順

  1. ウェイテッドフックをラインに結ぶ
  2. ワームの頭からフックを真っ直ぐ刺す
  3. ボディに針先を埋め込んでセット完了
12

ラバージグ

鉛やタングステンのヘッドにラバースカートとフックが一体化したルアー。トレーラーワームを付けてカバー撃ちからスイミングまで幅広く使える。ヘッド形状（フットボール・アーキー・ラウンド）によって得意な状況が異なり、ウエイトとスカートカラーの選択がキモ。

必要パーツ

  • ラバージグ（3.5g - 14g）
  • トレーラーワーム（ホグ系・クロー系・チャンク）
  • トレーラーフック（任意）

有効な状況

カバー撃ち（アーキーヘッド）、ディープのボトムバンプ（フットボールヘッド）、スイミング（ラウンドヘッド）。通年使える万能ルアーだが、特に低水温期に強い。

セッティング手順

  1. ラバージグをラインに直結する
  2. トレーラーワームの頭からフックに差し込む
  3. ワームがジグヘッドにしっかり密着するようセットする
  4. 必要に応じてトレーラーフックを追加する
